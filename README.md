# MUI Demo

### Instructions
- Compile MUI5 library
```shell
git clone https://github.com/caiseL/st-material-ui.git
cd st-material-ui
git checkout fix-compile
sbt +publishLocal
```

- Clone this repository and run the demo
```shell
git clone https://github.com/caiseL/mui-demo.git
cd mui-demo
sbt dev 
```
